# ROS setup notes

This `setup` directory is split by host platform:

- `linux/`: Linux Mint 20.x / Ubuntu 20.04 + ROS 2 Foxy flow, reorganized from the referenced Naver Cafe post.
- `mac/`: macOS setup in two variants.

If you want the closest reproduction of the original post, start with `linux/`.
If you want a practical macOS route on Apple Silicon, start with `mac/conda/`.
